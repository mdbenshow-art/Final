# Final – 台灣農產品 3‑Sigma 市場分析

## 目標
本倉庫展示 **2025 年 6 月 12 日** 台灣 24 個農產品批發市場之 **3‑Sigma 控制限分析**，包括:
- 市場分區（北部 / 中部 / 南部）
- 各類別（水果、蔬菜、花卉）的均價、標準差、上下控制限
- 高價異常作物的辨識與經濟解讀
- 完整的 **白皮書**（`whitepaper.md`），約 2000 字
- 兩套視覺素材：**資訊圖表** (`infographic.png`) 與 **海報** (`poster.png`)

---

## 如何使用
1. **檢視白皮書**：在倉庫根目錄直接點擊 `whitepaper.md`（GitHub 會自動渲染為 Markdown）
2. **下載視覺素材**：點擊 `infographic.png` 或 `poster.png` 取得高解析度圖檔
